## Quickstart: Clone the Project
### First, clone this repository.
#### Then, in the terminal:

```
bash or (zsh for mac)
> bundle install
> bin/rails db:create
> bin/rails db:migrate

> bundle exec rspec # Run the tests to ensure it works
> bin/rails server # Start the server at localhost:3000
```
